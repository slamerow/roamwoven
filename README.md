# Roamwoven

Roamwoven turns messy trip materials into a clean, private, mobile-friendly trip app. Users pay per trip, upload confirmations/docs/screenshots/notes, answer clarification questions, and publish a private PWA-style traveler app.

## Current Status

This repo is at product scaffold stage. The key planning docs live in `docs/`:

- `docs/prd.md`
- `docs/technical-architecture.md`
- `docs/implementation-backlog.md`

## Local Development

```bash
npm install
npm run dev
```

## Initial Build Target

The first visible beta target is:

- Landing page with Asia app demo positioning.

- Trip dashboard and create-trip shell.

- Traveler app route at `/t/[token]`.
